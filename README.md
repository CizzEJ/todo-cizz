# todo-cizz

A simple to-do list CLI tool.

## Usage

```
python todo.py add "Buy groceries"
python todo.py list
python todo.py done 1
python todo.py remove 1
```

## Requirements

- Python 3.8+

## Installation

```
git clone https://github.com/your-username/todo-cizz.git
cd todo-cizz
python todo.py list
```
